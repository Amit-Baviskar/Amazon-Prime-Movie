# Amazon-Prime-Movie

Amazon Prime is another one of the most popular media and video streaming platforms. They have close to 10000 movies or tv shows available on their platform, as of mid-2021, 
they have over 200M Subscribers globally. This tabular dataset consists of listings of all the movies and tv shows available on Amazon Prime,
along with details such as - cast, directors, ratings, release year, duration, etc.*


# Entry In Table
  Show_id	- Distinct Show-id
  Type - Describe the Type Of Movie
  Title	- Title / Name Of TV Show / Movie
  Director - Name Of Director 
  Cast - Actor/Actoress Work In Content
  Country	- Country 
  Date_Added - Date On Which Content Added On Site
  Release_year - Date On Which Content Release  From Site
  Rating - Average Rating to Content 	
  Duration - Total Duration 
  Listed_In_Description - Description About Content
  
# Dataset - https://www.kaggle.com/datasets/shivamb/amazon-prime-movies-and-tv-shows
